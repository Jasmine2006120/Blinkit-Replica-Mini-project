# Blinkit-Replica-Mini-project
Blinkit Clone (HTML & CSS)
📌 Project Overview

This project is a front-end replica of the Blinkit website built using HTML and CSS. The goal of this project was to practice modern web design concepts, layout creation, and responsive user interface development.

The website mimics the basic structure and appearance of Blinkit's homepage, including a navigation sidebar and a product display section.

🚀 Features
Responsive and clean user interface
Sidebar navigation menu
Product display section showcasing various items
Modern layout inspired by Blinkit's design
Organized HTML structure and CSS styling
User-friendly design with smooth visual experience
🛠️ Technologies Used
HTML5
CSS3
